Cookbook

How can I write a comment in my code?

// this is a comment, it does nothing but
// it helps me understand what's happening

How can I get the width of the current document?

width = app.activeDocument.width;

How can I iterate over the layers in a document?

document = app.activeDocument;
for(var i=0; i<document.layers.length; i++) {
  // do something amazing here
}

How can I write conditional logic?

if(app.activeDocument.width > 500){
  // the document is greater than 500 pixels wide
} else{
  // the document is less than 500 pixels wide
}

How can I duplicate the selected path or layer?

copy = app.activeDocument.selection[0].duplicate();

How can I position a compound path relative to the art board?

selection = app.activeDocument.selection[0];
selection.position = [0, 0];

How can I position a compound path relative to itself?

// scoot the selection 50 pixels to the right
selection = app.activeDocument.selection[0];
selection.translate(50, 0);

How can I rotate a compound path with degrees?

// rotate the selection 90 degrees
selection = app.activeDocument.selection[0];
selection.rotate(90);

How can I use a function to generate a random number?

function random(min, max){
  return Math.floor(Math.random() * (max - min + 1)) + min;
}

How can I get an element by it's name?

document = app.activeDocument;
thing = document.pageItems.getByName('myname');
thing.opacity = 25;

How can I prompt a user for input?

name = prompt('What up playa?');

How can I add undo to my script?

app.undo();

How can I change the zOrder of a compound path?

// sends the selected path to the very back
app.activeDocument.selection[0].zOrder(ZOrderMethod.SENDTOBACK);

How can I get the yellow CMYK value of the 5th item in the swatch panel?

swatches = document.swatches;
yellow = swatches[4].color.yellow;

How can I update the text of a selected object?

app.activeDocument.selection[0].words.length;

Will the code listed here work in other Adobe applications (Photoshop, Bridge, inDesign, etc.)

For the most part, but different Adobe applications have different concepts. For example: selecting and manipulating paths will not work in Bridge. Pages are a concept within inDesign which has no equal counterpart in Illustrator of Photoshop.
